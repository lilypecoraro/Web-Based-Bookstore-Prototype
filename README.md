<h1 align="center">Web-Based Bookstore Prototype</h1>

<p align="center">
<strong>A full-stack ASP.NET Core MVC bookstore prototype designed to explore e-commerce flows, data modeling, and user experience.</strong>
<br/>
Serves as an early-stage prototype for the larger <em>Bevo’s Books</em> platform.
</p>

---

<h2>About</h2>

This project is a web-based bookstore prototype built using ASP.NET Core MVC. It models the core functionality of an online bookstore, including product browsing, database-backed content, and dynamic page rendering.

The prototype focuses on establishing a clean architecture, clear separation of concerns, and scalable foundations for a larger e-commerce system.

---

<h2>Goal & Requirements</h2>

<strong>Primary objectives:</strong>
<ul>
  <li>Design a functional bookstore using the MVC pattern</li>
  <li>Model books, categories, and related data in a relational database</li>
  <li>Implement dynamic views using Razor and server-side rendering</li>
  <li>Lay groundwork for authentication, roles, and expanded commerce features</li>
</ul>

<strong>Constraints:</strong>
<ul>
  <li>Prototype-level scope (not production commerce)</li>
  <li>Focus on architecture and extensibility over visual polish</li>
  <li>Maintain clear separation between data, logic, and presentation layers</li>
</ul>

---

<h2>Key Learnings</h2>

<ul>
  <li>Building scalable ASP.NET Core MVC applications</li>
  <li>Structuring controllers, models, views, and data access layers</li>

<h2>Running the Project</h2>

<ol>
  <li>Clone the repository</li>
  <li>Open the solution in Visual Studio</li>
  <li>Configure database connection in <code>appsettings.json</code></li>

</ol>
